youdao
======

Youdao English to Chinese translation in terminal.

# Install #
    npm install -g youdict

# Usage #
    $ youdict hello
    hello [hə'ləʊ; he-][hɛˈlo, hə-]

    int. 喂；哈罗
    n. 表示问候， 惊奇或唤起注意时的用语
    n. (Hello)人名；(法)埃洛

    Hello, who's speaking, please?
    喂， 请问你是谁呀?

    Hello! This is little Li speaking.
    喂， 我是小李。

    “Hello?” he said.
    “你好吗？” 他说。
